# Website

This website is built using [VitePress](https://vitepress.dev/).

## Installation

```bash
yarn
```

## Local Development

```bash
yarn start
```

This command builds and serves all locales together, so both `http://localhost:5173/` (English) and `http://localhost:5173/vi/` (Vietnamese) are available.

## Build

```bash
yarn build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.
